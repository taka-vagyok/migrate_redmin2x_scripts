migrate_redmin2x_scripts
================

# Description
It is bash script for migrate redmine from 2.x to 2.x.

# Required
bash 

# Environment
* FreeBSD 8.2

# How to use
bash migrate_redmine2x_scripts/migrate_redmine2x.bash <old_redmine_path> <new_redmine_path>
